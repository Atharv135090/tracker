# 🚀 PrepTrack

A modern placement preparation dashboard built with **HTML, CSS, and JavaScript** to help students track learning progress, build consistency, and stay accountable throughout their placement journey.

🌐 **Live Demo:** https://atharv135090.github.io/tracker/

---

## ✨ Features

### 📚 Learning Progress Tracking

Track progress across:

* Data Structures & Algorithms (DSA)
* Web Development
* Spoken English

Features:

* ✅ Mark lectures complete
* 🔄 Unmark lectures anytime
* 📈 Real-time progress updates
* ➕ Add custom lectures
* 💾 Auto-save progress

---

### 📊 Analytics Dashboard

Monitor your preparation with:

* 🎯 Overall Completion %
* 📖 Total Lectures
* ⏳ Remaining Lectures
* 📅 Weekly Progress
* ⚡ Productivity Score
* 🔥 Current Streak

---

### 🗓️ GitHub-Style Activity Heatmap

Visualize consistency through a contribution-style heatmap.

* Daily activity tracking
* Year selector
* Active day count
* Streak calculations
* Contribution insights

---

### 🏆 Achievement System

Unlock milestones as you progress.

Examples:

* 🎉 First Lecture Completed
* 📚 25 Lectures Completed
* 🚀 50 Lectures Completed
* 👑 100 Lectures Completed
* 🔥 7-Day Streak

---

### ⏱️ Study Timer

Built-in focus timer with:

* 25 Minute Focus
* 45 Minute Deep Work
* 60 Minute Long Session
* Custom Session Duration
* Daily Study Tracking

---

### 🎨 Modern UI

Inspired by:

* Linear
* Stripe
* Raycast
* Vercel
* Notion

Features:

* 🌙 Dark Mode
* ☀️ Light Mode
* 📱 Responsive Design
* ✨ Smooth Animations
* 🎯 Modern Dashboard Experience
